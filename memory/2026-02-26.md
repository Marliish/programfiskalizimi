### 7. âś… Decimal/String Type Errors

## Architecture Notes

### API Structure
- Auth: `/v1/auth/*`
- POS transactions: `/v1/pos/*` (not `/v1/transactions`)
- Products: `/v1/products`
- Categories: `/v1/categories`
- JWT required for all authenticated endpoints

### Schema Validation Pattern
- Query params from URLs are strings
- Use `z.preprocess()` or helper functions for type coercion
- Keep reusable helpers in `schemas/helpers.ts`

### Database Migrations
- Migrations can fail mid-apply
- Use `prisma migrate resolve` to mark as applied/rolled-back
- Always regenerate client after schema changes
- Check actual DB schema with `psql` when in doubt
