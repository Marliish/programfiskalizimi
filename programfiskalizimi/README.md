## 📊 **COMMUNICATION PROTOCOL**

### **Daily Reports (Every Day at 6:00 PM)**
Turi sends you on Telegram:
- ✅ What was completed today
- 🔄 What's in progress
- 🚧 Any blockers
- 📅 Plan for tomorrow

### **Weekly Reports (Every Friday)**
- Summary of the week
- Demos of completed features
- Metrics (code commits, features done)
- Next week's plan

### **You Talk to Turi Only**
- Ask: "What's the progress?"
- Ask: "What did backend dev finish?"
- Ask: "Show me what designer created"
- Give instructions: "Tell backend to prioritize X"
- Turi coordinates with all agents
