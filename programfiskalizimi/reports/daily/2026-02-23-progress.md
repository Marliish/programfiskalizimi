### **You Can Run:**

```bash
# 1. Start Docker services
cd programfiskalizimi/code/fiscalnext-monorepo/infrastructure/docker
docker-compose up -d

# This starts:
# ✅ PostgreSQL (port 5432)

npm install
npm run dev

# Server starts on http://localhost:5000
# Try: curl http://localhost:5000/health
```
