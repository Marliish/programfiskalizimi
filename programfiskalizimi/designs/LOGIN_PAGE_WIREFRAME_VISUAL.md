## ⌨️ KEYBOARD NAVIGATION (Tab Order)

```
1. Email input           [TAB]
2. Password input        [TAB]
3. Show/hide password    [TAB]
4. Remember me checkbox  [TAB]
5. Forgot password link  [TAB]
6. Login button          [TAB] [ENTER to submit]
7. Google button         [TAB]
8. Sign up link          [TAB]

[ENTER] from any input = Submit form
[ESC] = Clear form (optional)

All elements have 3px focus ring (Primary 500)
```
