# FiscalNext User Guide

## Welcome to FiscalNext!

FiscalNext is a comprehensive fiscal management and Point of Sale (POS) system designed for businesses in Albania and Kosovo that need to comply with fiscal regulations.

## Getting Started

### First Login

1. Navigate to the login page
2. Enter your email and password
3. If this is your first time, you'll be prompted to change your password
4. Set up two-factor authentication (recommended)

### Dashboard Overview

The dashboard is your central hub, showing:

- **Today's Sales**: Real-time sales figures
- **Active Orders**: Orders currently being processed
- **Low Stock Alerts**: Products running low on inventory
- **Recent Activity**: Latest transactions and updates

## Managing Products

### Adding a New Product

1. Navigate to **Products** → **Add Product**
2. Fill in the required fields:
   - **Name**: Product name
   - **SKU**: Unique product code
   - **Price**: Selling price (including tax)
   - **Category**: Product category
   - **Tax Rate**: Applicable tax rate (default: 20%)
3. Optional fields:
   - **Description**: Detailed product description
   - **Barcode**: Barcode for scanning
   - **Images**: Product photos
4. Click **Save** to create the product

### Editing Products

1. Go to **Products**
2. Click on the product you want to edit
3. Make your changes
4. Click **Update**

### Managing Inventory

1. Navigate to **Inventory**
2. View stock levels across all locations
3. **Adjust Stock**: Add or remove inventory
4. **Transfer Stock**: Move inventory between locations
5. Set **Reorder Points** to get automatic low stock alerts

## Processing Sales

### POS System

1. Navigate to **POS** or use the dedicated POS terminal
2. **Add Products**:
   - Search by name
   - Scan barcode
   - Browse categories
3. **Adjust Quantities**: Click +/- buttons
4. **Apply Discounts**: Percentage or fixed amount
5. **Customer**: Add customer info (optional)
6. **Payment**:
   - Cash
   - Card
   - Bank Transfer
7. **Complete Sale**: Generate fiscal receipt

### Processing Returns

1. Find the original order
2. Click **Refund**
3. Select items to return
4. Specify reason
5. Process refund
6. Fiscal credit note is generated automatically

## Customer Management

### Adding Customers

1. Navigate to **Customers** → **Add Customer**
2. Enter customer details:
   - Name
   - Email
   - Phone
   - Address
3. Optional: Add to loyalty program
4. Click **Save**

### Loyalty Programs

- Customers earn points with each purchase
- Redeem points for discounts
- View point balance in customer profile
- Set up custom reward tiers in **Settings**

## Troubleshooting

### Common Issues

#### "Cannot connect to fiscal authority"

1. Check internet connection
2. Verify fiscal credentials in **Settings** → **Fiscal Settings**
3. Contact support if issue persists

#### "Low stock alert not showing"

1. Go to **Products**
2. Check if reorder point is set
3. Verify notification settings

#### "Receipt not printing"

1. Check printer connection
2. Verify printer settings in **Settings** → **Printers**
3. Test print from printer settings

#### "Login failed"

1. Verify email and password
2. Check if account is locked (5 failed attempts)
3. Use "Forgot Password" to reset
4. Wait 15 minutes if locked

## Frequently Asked Questions

**Q: How do I add multiple locations?**  
A: Go to Settings → Locations → Add Location. Each location can have separate inventory and staff.

**Q: Can I customize receipt templates?**  
A: Yes! Go to Settings → Receipt Templates to customize the layout, logo, and footer text.

**Q: How do I export data?**  
A: Most reports have an "Export" button. Choose PDF, Excel, or CSV format.

**Q: Is my data backed up?**  
A: Yes, automatic daily backups are performed. You can also manually backup from Settings → Backup.

**Q: How do I add more users?**  
A: Settings → Users → Add User. Assign appropriate roles (Admin, Manager, Cashier).

**Q: What's the difference between roles?**  
A:
- **Admin**: Full system access
- **Manager**: Everything except system settings
- **Cashier**: POS and basic inventory viewing
