───────────────────────────────────────────────────────────────
🔄 WORKFLOW EXAMPLE
───────────────────────────────────────────────────────────────

1. Host assigns party to Table 5
   → Table status: Available → Occupied

2. Server creates order on tablet
   → OrderEditor: Add items, modifiers, courses

3. Server sends order to kitchen
   → KitchenDisplay: Order appears in "Pending"

4. Kitchen staff starts cooking
   → Status: Pending → Preparing

5. Kitchen marks order ready
   → Status: Preparing → Ready (pulse animation)

6. Server delivers food & bumps order
   → Status: Ready → Served

7. Customer pays & adds 18% tip
   → TipEntryModal: Quick preset selected

8. Order complete, table freed
   → Table status: Occupied → Available
